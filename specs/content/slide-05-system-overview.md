## 본문
수집된 경험을 바탕으로 시스템이 스스로 분류 기준을 세우고 탐색 체계를 구축합니다.
단편적인 대화는 구조화된 데이터로 치환되어 영구 보존됩니다.

## 다이어그램
원본 데이터(대화 수집) → AI 가공(요약 분류) → 구조화된 지식(인덱싱)

## 팝업
- **Knowledge Engineering**: 단편적인 String 형태의 대화를 구조화된 JSON 과 Markdown 으로 치환하여 영구 보존하는 과정입니다.
