AI 의 3 층 기억

AI 도 3 가지 기억 방식을 활용합니다

L1: 사전 학습 모델 (Pre-trained) - 변경 불가 고정 지식
L2: 에이전트 메모리 (File System) - 실시간 추가 개인 메모리 ?
L3: 문서 참조 (RAG) - 외부 연동 참조 자료 ?

L1: Model

학습 시점: 2024 년
데이터량: 수 TB

L2: Memory ← 주제

학습 시점: 실시간
데이터량: 수 MB~GB

L3: Retrieval

학습 시점: 필요 시
데이터량: 수 GB~TB

💡Tip: 오늘은 L2 '에이전트 메모리' 에 해당하는 파일 기반 지식 시스템을 다룹니다.

중요도

동적 점수에 따른 3 단계 분류

T1: 핵심 규칙

항상 참조해야 하는 행동 규칙
점수: 80~100

T2: 일상 지식

특정 상황에 꺼내보는 지식
점수: 40~79

T3: 보조 정보

요청 시 제공되는 배경 정보
점수: 0~39

점수 계산?

우선순위 + 최신성 + 사용빈도 + 허브노드

감가상각?

90 일 이상 미사용 시 점수 감소

💡Tip: 점수 계산과 감가상각 알고리즘을 통해 불필요한 Context 의 메모리 점유를 방지합니다.
