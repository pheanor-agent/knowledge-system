중요도

동적 점수에 따른 3 단계 분류

T1: 핵심 규칙

항상 참조해야 하는 행동 규칙
점수: 80~100

T2: 일상 지식

특정 상황에 꺼내보는 지식
점수: 40~79

T3: 보조 정보

요청 시 제공되는 배경 정보
점수: 0~39

점수 계산?

우선순위 + 최신성 + 사용빈도 + 허브노드

감가상각?

90 일 이상 미사용 시 점수 감소

💡Tip: 점수 계산과 감가상각 알고리즘을 통해 불필요한 Context 의 메모리 점유를 방지합니다.
