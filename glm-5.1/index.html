AI 의 3 층 기억

AI 도 3 가지 기억 방식을 활용합니다

L1: 사전 학습 (Pre-trained)
학습 시점: 2024 년 / 데이터량: 수 TB
L2: 에이전트 메모리 (File System)?
학습 시점: 실시간 / 데이터량: 수 MB~GB
L3: 문서 참조 (RAG)?
학습 시점: 필요 시 / 데이터량: 수 GB~TB
💡Tip: L1 이 없으면 기초 언어 이해가 불가능하고, L2 가 없으면 실시간 상황 판단이 불가능하며, L3 가 없으면 전문 분야 심화 분석이 불가능합니다. 오늘은 L2 '에이전트 메모리' 를 다룹니다.

점수 계산?

우선순위 + 최신성 + 사용빈도 + 허브노드

감가상각?

90 일 이상 미사용 시 점수 감소

참조 방식?

T1: 매 대화 자동 로드 / T2: 키워드 검색 시 로드 / T3: 명시적 요청 시만 로드

💡Tip: 점수 계산과 감가상각 알고리즘을 통해 불필요한 Context 의 메모리 점유를 방지합니다.
