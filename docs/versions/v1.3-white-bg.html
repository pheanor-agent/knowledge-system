챗봇의 기억 상실

대화를 마치면 모든 내용이 사라집니다.
"어제 말했던 프로젝트 이름이 뭐였더라?"

→ 파일을 활용하면 기억을 남길 수 있습니다
비유

신입사원이 자료를 정리하듯

👤

신입사원

방대한 자료를 접하면, 폴더를 만들어 정리합니다. 중요한 문서에는 태그를 붙이고, 검색하기 쉬운 구조를 만듭니다.

🤖

AI 에이전트

수만 건의 데이터를 처리합니다. 파일을 체계적으로 저장하고, 검색할 수 있는 구조를 만들 때 비로소 기억이 됩니다.

→ AI도 체계적인 파일 정리가 필요합니다
AI의 3층 기억

AI가 지식을 다루는 3가지 방식

1

모델 학습 (先天적)

LLM이出厂時 가지고 있는 지식 — 세계사, 과학, 언어

2

에이전트 메모리 (后天적) ← 오늘 주제

파일로 저장하는 지식 — 프로젝트 기록, 사용자 선호

3

RAG (외부 검색)

인터넷에서 실시간 검색 — 최신 뉴스, 날씨

💡Tip: 오늘 세미나는 "에이전트 메모리"를 다룹니다. AI가 파일을 어떻게 저장하고 검색하는지 소개합니다.
